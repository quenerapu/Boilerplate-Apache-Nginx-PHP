# Boilerplate (Apache o Nginx) + PHP

Just add water.

## ¿Que quieres un servidor Apache + PHP?

`podman compose --profile apache up -d`  
`podman compose --profile apache down`

## ¿Que quieres un servidor Nginx + PHP-FPM?

`podman compose --profile nginx up -d`  
`podman compose --profile nginx down`

* Pon `docker` en lugar de `podman` si es `docker` lo que usas.
